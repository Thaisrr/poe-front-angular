Attribute / Property Binding

On peut passer dynamiquement des données du Ts vers les attributs HTML avec la syntaxe []

Les attributs html basique attendent une string, en dur, quand ils sont entre [], ils attendent du JS à interpréter :

  • src="monlien.jpeg"
  • [src]="du js qui retourne une string"

Cette syntaxe est vrai pour tous les attributs HTML, même les attributs personnalisés.

Data Binding : classes

On peut gérer les classes dynamiquement de plusieurs manières :

Les syntaxes de [class]

  • [class]="du_js qui retourne un string"
  • [class]="(condition)? 'c_true' : 'c_false'"
  • [class.nom_classe]="condition" => l'élément à la classe "ma_classe" si condition est true

Les syntaxes de [ngClass]

On peut utiliser la directive ngClass pour gérer les classes dynamiquement. Elle attend comme valeur :

  • String ( comme [class] ) : [ngClass]="ma_classe"
  • String[] [ngClass="[]"
  • Object : [ngClass]="{{"{ma_classe: condition}"}}"
