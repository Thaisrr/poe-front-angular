Les Pipes

Les pipes sont des paramètres / des fonctions qui servent à formater les données affichées dans le template.

Les pipes de texte

  • {{message}}
  • {{ message | uppercase }}
  • {{ message | lowercase }}
  • {{message | titlecase}}

Date

  • {{today}}
  • {{today | date}}
  • Short : {{today | date : 'short'}}
  • Medium : {{today | date: 'medium'}}
  • Long: {{today | date: 'long'}}
  • Full: {{today | date: 'full'}}
  • Numéro dans le mois ( 1 à 31 ) : d {{today | date: 'd'}}
  • Numéro dans la semaine ( 1 à 7 ): c {{today | date: 'c'}}
  • Abréviation du jour: ccc {{today | date: 'ccc'}}
  • Jour complet: cccc {{today | date: 'cccc'}}
  • Numéro de mois ( 1 - 12 ) : M {{today | date: 'M'}}
  • Numéro de mois ( 01 - 12 ) : MM {{today | date: 'MM'}}
  • Abréviation du mois : MMM {{today | date: 'MMM'}}
  • Nom du mois : MMMM {{today | date: 'MMMM'}}
  • Année {{today | date: 'yy'}}
  • Année {{today | date: 'yyyy'}}
  • Date : {{today | date: 'cccc, d, MMMM, yyyy'}}
  • Date : {{today | date: 'd/MM/yyyy'}}
  • Heure ( 1, 12) {{today | date: 'h'}}
  • Heure ( 01, 12) {{today | date: 'hh'}}
  • Heure ( 0, 23) {{today | date: 'H'}}
  • Heure ( 00, 23) {{today | date: 'HH'}}
  • Minute ( 0 - 59 ) {{today | date: 'm'}}
  • Minute ( 00 - 59 ) {{today | date: 'mm'}}
  • {{today | date: 'HH:mm'}}
  • {{today | date: 'h:mm a'}}
